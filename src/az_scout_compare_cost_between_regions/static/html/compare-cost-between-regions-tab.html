Required
Usage Details / Cost Export

The usage details export contains all billed line items including per-resource, per-day usage and cost data for a single billing month.

  1. Sign in to the Azure Portal
  2. Search for Cost Management + Billing
  3. Select the billing scope for your EA billing account
  4. Select Usage + charges
  5. For the desired month, select Download
  6. Under Usage Details, select charge type and click Prepare document
  7. When ready, click Download csv
EA usage download documentation
  1. Sign in to the Azure Portal
  2. Search for Cost Management + Billing
  3. Select a billing scope (billing profile)
  4. On the overview page, under Shortcuts, click Download usage and prices
  5. Under Azure usage file, select Prepare
  6. When ready, click Download
MCA usage download documentation
Optional
Price Sheet

The price sheet contains your negotiated per-meter prices across all Azure regions. Used for cross-region cost comparison in Step 3.

  1. In Cost Management + Billing, select your EA billing scope
  2. Go to Usage + charges
  3. For the same month, click Download
  4. Under Price Sheet, click Prepare document
  5. When ready, click Download Azure price sheet
  1. In Cost Management + Billing, go to Invoices
  2. Find the invoice row for the billing period
  3. Click the ellipsis (…) at the end of the row
  4. Select Azure price sheet
  5. The file downloads automatically when ready
